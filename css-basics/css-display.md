# CSS Display

Used to control layouts by specifying how an element should be displayed.

**Visibility** determines if an element is to be showed on not.

The following are the available display values:

* `Display: None;`
* `Display: inline;`
* `Display: block;`
* `Visibility: hidden;` 

Examples:

```text
li {
    display: inline;
}

h1 {
    visibility: hidden;
}
```
